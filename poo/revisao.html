Revisão de POO em C#

Projeto de Revisão

Projeto: Sistema de Cadastro de Biblioteca

Neste exercício, você deverá criar um pequeno sistema em C# para revisar os principais conceitos de Programação Orientada a Objetos.

Objetivos da atividade

  • Criar classes em C#
  • Definir atributos e propriedades
  • Criar métodos
  • Instanciar objetos no Program
  • Chamar métodos dos objetos criados
  • Alterar valores utilizando propriedades

Enunciado

Crie um sistema de Biblioteca em C#. O sistema deverá possuir uma classe chamada Livro.

A classe Livro deve possuir:

Propriedades:

  • Titulo
  • Autor
  • AnoPublicacao
  • QuantidadePaginas
  • Emprestado

Métodos:

  • ExibirInformacoes() → mostra os dados do livro
  • Emprestar() → altera o status do livro para emprestado
  • Devolver() → altera o status do livro para disponível
  • EstaDisponivel() → retorna se o livro está disponível ou não

O que o Program deve fazer

  1. Criar pelo menos 2 objetos da classe Livro
  2. Preencher os dados dos livros
  3. Exibir as informações na tela
  4. Emprestar um dos livros
  5. Mostrar novamente as informações
  6. Devolver o livro
  7. Exibir o status final

Exemplo de saída esperada

Título: O Hobbit
Autor: J.R.R. Tolkien
Ano: 1937
Páginas: 310
Status: Disponível

Emprestando livro...

Título: O Hobbit
Autor: J.R.R. Tolkien
Ano: 1937
Páginas: 310
Status: Emprestado

Regras

  • Faça a classe separadamente.
  • Depois use a classe no Program.cs.
  • Não use herança.
  • Foque em atributos, propriedades, métodos e objetos.

Desafio extra

Após terminar, faça uma validação para impedir que um livro já emprestado seja emprestado novamente.
