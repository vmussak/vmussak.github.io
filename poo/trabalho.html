Objetivo

Criar um programa simples em C# para praticar os conceitos iniciais de Programação Orientada a Objetos.

  • Criar classes
  • Definir propriedades
  • Criar métodos
  • Instanciar objetos no Program.cs
  • Chamar métodos dos objetos criados

Enunciado

Crie um sistema simples de cadastro de produtos. O sistema deverá possuir uma classe chamada Produto.

A classe Produto deve possuir:

Propriedades:

  • Nome: string
  • Preco: double
  • Categoria: string

Métodos:

  • ExibirInformacoes(): void → mostra os dados do produto
  • AplicarDesconto(double percentual): void → reduz o preço do produto conforme o percentual informado

O que o Program deve fazer

  1. Criar pelo menos 2 objetos da classe Produto
  2. Preencher os dados dos produtos
  3. Exibir as informações na tela
  4. Aplicar desconto em um dos produtos
  5. Mostrar novamente as informações atualizadas

Exemplo de saída esperada

Produto: Caderno
Categoria: Papelaria
Preço: R$ 25,00

Aplicando desconto de 10%...

Produto: Caderno
Categoria: Papelaria
Preço: R$ 22,50

Regras

  • Faça a classe separadamente.
  • Depois utilize a classe no Program.cs.
  • Não use herança.
  • Foque em classes, propriedades, métodos e objetos.
  • O programa pode ser feito em Console App.

Pesquisa

Responda brevemente:

  1. O que é uma classe?
  2. O que é uma instância (objeto)?
  3. Qual a diferença entre propriedade e método?
  4. Para que servem os tipos em C#?

Desafio extra

Crie um terceiro produto e aplique descontos diferentes em cada um deles.
