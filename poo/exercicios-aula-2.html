Exercícios de Lógica de Programação com C# (Nível Fácil)

1) Tipos Primitivos e Variáveis

  1. Declare uma variável do tipo int com o valor 10 e imprima na tela.
  2. Declare uma variável do tipo double com valor 5.5 e imprima o dobro do valor.
  3. Declare uma variável do tipo string com seu nome e exiba a mensagem: "Olá, [nome]".
  4. Declare uma variável do tipo bool com valor true e imprima o valor invertido.

2) Operadores e Expressões

  1. Crie duas variáveis inteiras e imprima: soma, subtração, multiplicação e divisão.
  2. Crie uma variável inteira e verifique se ela é maior que 100. Imprima o resultado (true ou false).

3) Estrutura Condicional if / else

  1. Leia um número e verifique se ele é positivo ou negativo.
  2. Leia a idade de uma pessoa e informe se ela é maior de idade (18 anos ou mais).
  3. Leia um número e verifique se ele é par ou ímpar.
  4. Leia duas notas, calcule a média e informe se o aluno foi aprovado (média >= 6).

4) Estrutura switch

  1. Leia um número de 1 a 7 e imprima o dia da semana correspondente usando switch.
  2. Leia uma letra (A, B ou C) e imprima uma mensagem diferente para cada caso.

5) Laço for

  1. Use um for para imprimir os números de 1 a 10.
  2. Use um for para imprimir apenas os números pares de 1 a 20.
  3. Use um for para calcular a soma dos números de 1 a 100.

6) Laço while

  1. Use um while para imprimir os números de 1 a 5.
  2. Crie um programa que peça uma senha e continue pedindo enquanto a senha estiver incorreta.

7) Laço foreach

  1. Crie um array com 5 números e use foreach para imprimir cada número.
  2. Crie uma lista de nomes e use foreach para imprimir uma saudação para cada nome.

8) Exercício Integrador

  1. Crie um programa que:
    • Peça um número ao usuário
    • Use if para verificar se é positivo
    • Use for para imprimir os números de 1 até esse número

9) Contador com validação (while + if)

  1. Leia um número inteiro N (N > 0).
    • Se o usuário digitar N <= 0, continue pedindo até informar um valor válido.
    • Depois imprima todos os números de N até 1 (decrescente) usando while.

10) Menu de operações (switch + loop)

  1. Crie um menu repetitivo com as opções:
    • 1: Somar
    • 2: Subtrair
    • 3: Multiplicar
    • 4: Dividir
    • 0: Sair
    Para as opções 1–4, peça dois números e mostre o resultado. Trate divisão por zero (se o segundo número for 0, exiba uma mensagem e não divida). O menu deve continuar até o usuário escolher 0.

11) Contar vogais em uma string (foreach + if)

  1. Leia uma frase do usuário. Use foreach para percorrer os caracteres e conte quantas vogais existem (a, e, i, o, u), ignorando maiúsculas/minúsculas. Ao final, imprima a contagem total e a contagem individual por vogal.

12) Maior, menor e média de uma sequência (for + if)

  1. Leia um número N (quantidade de valores). Depois leia N números inteiros e calcule e imprima: maior, menor e média (como double).
    • Requisito: não use LINQ (Max, Min, Average) — faça "na mão".

13) Sequência de Fibonacci até um limite (while)

  1. Leia um inteiro L (L >= 1). Gere e imprima a sequência de Fibonacci enquanto o próximo valor for <= L.
    • Exemplo: se L = 20, imprime: 0 1 1 2 3 5 8 13

    Console.WriteLine("Positivo");

4) Estrutura switch

// dia da semana
int dia = 3;
switch (dia)
{
    case 1: Console.WriteLine("Segunda"); break;
    case 2: Console.WriteLine("Terça");   break;
    case 3: Console.WriteLine("Quarta");  break;
    case 4: Console.WriteLine("Quinta");  break;
    case 5: Console.WriteLine("Sexta");   break;
    case 6: Console.WriteLine("Sábado");  break;

}

Escolheu B

10) Menu de operações (switch + loop)

int opcao;
do
{
    Console.WriteLine("\n1-Somar  2-Subtrair  3-Multiplicar  4-Dividir  0-Sair");
    Console.Write("Opção: ");
    opcao = int.Parse(Console.ReadLine());

    if (opcao >= 1 && opcao <= 4)
    {
        Console.Write("Primeiro número: ");
        double a = double.Parse(Console.ReadLine());
        Console.Write("Segundo número: ");
        double b = double.Parse(Console.ReadLine());

        switch (opcao)
        {
            case 1: Console.WriteLine("Resultado: " + (a + b)); break;
            case 2: Console.WriteLine("Resultado: " + (a - b)); break;
            case 3: Console.WriteLine("Resultado: " + (a * b)); break;
            case 4:
                if (b == 0)
                    Console.WriteLine("Erro: divisão por zero!");
                else
                    Console.WriteLine("Resultado: " + (a / b));
                break;
        }
    }
} while (opcao != 0);

Console.WriteLine("Até logo!");

11) Contar vogais (foreach + if)

Console.Write("Digite uma frase: ");
string frase = Console.ReadLine().ToLower();

int total = 0, cA = 0, cE = 0, cI = 0, cO = 0, cU = 0;

foreach (char c in frase)
{
    switch (c)
    {
        case 'a': cA++; total++; break;
        case 'e': cE++; total++; break;
        case 'i': cI++; total++; break;
        case 'o': cO++; total++; break;
        case 'u': cU++; total++; break;
    }
}

Console.WriteLine("Total de vogais: " + total);
Console.WriteLine($"a={cA}  e={cE}  i={cI}  o={cO}  u={cU}");

12) Maior, menor e média (for + if)

Console.Write("Quantidade de valores: ");
int n = int.Parse(Console.ReadLine());

int maior = int.MinValue, menor = int.MaxValue, soma = 0;

for (int i = 0; i < n; i++)
{
    Console.Write("Valor: ");
    int v = int.Parse(Console.ReadLine());
    soma += v;
    if (v > maior) maior = v;
    if (v < menor) menor = v;
}

Console.WriteLine("Maior: " + maior);
Console.WriteLine("Menor: " + menor);
Console.WriteLine("Média: " + (double)soma / n);

13) Fibonacci até um limite (while)

Console.Write("Digite o limite L: ");
int L = int.Parse(Console.ReadLine());

int a = 0, b = 1;
while (a <= L)
{
    Console.Write(a + " ");
    int temp = a + b;
    a = b;
    b = temp;
}
Console.WriteLine();

Exemplo (L = 20):

0 1 1 2 3 5 8 13
