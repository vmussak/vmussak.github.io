Projeto: Calculadora de Frete

Objetivo

Criar uma aplicação simples em Windows Forms com C# para calcular o valor de um frete, utilizando Programação Orientada a Objetos, herança e polimorfismo.

Protótipo da Interface

Descrição do Sistema

O usuário deverá informar o nome do cliente, o peso do pacote e o tipo de entrega. Ao clicar no botão de calcular, o sistema deverá mostrar o valor do frete.

Requisitos

  • O sistema deve possuir uma classe base chamada Entrega.
  • A classe Entrega deve conter as propriedades Cliente e Peso.
  • A classe Entrega deve possuir um método para calcular o frete.
  • Devem existir duas classes filhas: EntregaNormal e EntregaExpressa.
  • A entrega normal deve calcular o frete multiplicando o peso por 5.
  • A entrega expressa deve calcular o frete multiplicando o peso por 10.
  • O cálculo deve usar polimorfismo.

Campos da Tela

  • Nome do cliente
  • Peso do pacote
  • Tipo de entrega: Normal ou Expressa
  • Botão Calcular Frete
  • Área para exibir o resultado

Resultado Esperado

Após o cálculo, o sistema deve exibir:

  • Nome do cliente
  • Peso informado
  • Tipo de entrega escolhido
  • Valor final do frete
