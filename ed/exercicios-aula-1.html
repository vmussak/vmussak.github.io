Exercícios simples de lógica (Python)

1) Variáveis e Tipos

  1. Crie três variáveis: uma do tipo inteiro, uma do tipo float e uma do tipo string. Em seguida, imprima o tipo de cada uma usando type().
  2. Crie a variável x = 0 e imprima o resultado de bool(x).
  3. Crie a variável x = -10 e imprima o resultado de bool(x).

2) Operações Numéricas

  1. Crie duas variáveis inteiras a e b. Mostre: soma (a+b), subtração (a-b), multiplicação (a*b) e divisão (a/b).
  2. Crie uma variável float com 2 casas decimais e multiplique por 3. Mostre o resultado.
  3. Crie um número complexo e imprima o valor e o tipo usando type().

3) Booleanos e Operadores Lógicos

  1. Crie as variáveis idade = 18 e tem_carteira = True. Verifique se a pessoa pode dirigir usando idade >= 18 AND tem_carteira.
  2. Crie variáveis para idade e se está acompanhada. Verifique se pode entrar usando OR: pode entrar se for maior de idade OU estiver acompanhada.
  3. Dado x = 5, verifique se x > 2 AND x < 10. Imprima o resultado.
  4. Crie uma variável booleana (por exemplo, ativo = True) e imprima o valor invertido usando NOT.

4) Strings

  1. Crie duas strings e concatene usando o operador +. Imprima o resultado.
  2. Crie uma string e use o operador * para repetí-la 5 vezes. Imprima o resultado.
  3. Verifique se a palavra "python" está contida na string "estou aprendendo python" usando in. Imprima o resultado.

5) Range

  1. Crie um range que vá de 0 até 10 (sem incluir o 10). Converta para lista e imprima.
  2. Crie um range que vá de 1 até 20, pulando de 2 em 2. Converta para lista e imprima.
  3. Use um laço for com range(5) para imprimir os números de 0 a 4.

6) Listas

  1. Crie uma lista contendo: um número, uma string e um booleano. Imprima a lista.
  2. Crie uma lista com 5 números. Depois: adicione um novo número e remova um número existente. Imprima a lista final.
  3. Verifique se o número 10 está dentro de uma lista usando o operador in. Imprima o resultado.
  4. Dada a lista [1, 2, 3], altere o valor 2 para 20 e imprima a lista resultante.

Extra

  1. Crie um algoritmo que leia um número e imprima "Positivo" se ele for maior que zero, senão imprima "Não positivo".

If / Else / Elif

  1. Crie uma variável nota com um valor entre 0 e 10. Usando if, elif e else, imprima:
    • "Aprovado" se a nota for maior ou igual a 7
    • "Recuperação" se a nota for entre 5 (inclusive) e 7 (exclusive)
    • "Reprovado" se a nota for menor que 5
  2. Crie uma variável hora com um valor inteiro de 0 a 23. Usando if, elif e else, imprima:
    • "Bom dia" se for entre 6 e 11
    • "Boa tarde" se for entre 12 e 17
    • "Boa noite" se for entre 18 e 23
    • "De madrugada" caso contrário

Match / Case (Switch)

  1. Crie uma variável dia com um número de 1 a 7. Usando match/case, imprima o nome do dia da semana correspondente (1 = Segunda, 2 = Terça, ... 7 = Domingo). Para qualquer outro valor, imprima "Dia inválido".
  2. Crie uma variável operacao com um dos valores: "soma", "subtracao", "multiplicacao" ou "divisao". Crie também as variáveis a = 10 e b = 4. Usando match/case, calcule e imprima o resultado da operação correspondente. Para qualquer outro valor, imprima "Operação desconhecida".

Forzinho

  1. Dada a lista numeros = [3, 7, 2, 9, 4, 6, 1, 8, 5, 10], use um laço for combinado com if para:
    • Imprimir apenas os números pares
    • Calcular e imprimir a soma de todos os números ímpares
