Exercícios: calcular Big O

Para cada algoritmo abaixo, analise a complexidade de tempo em Big O em função de n. Considere n como o tamanho da entrada (ex.: tamanho da lista, valor de n, etc.). Ignore I/O (prints e inputs) e foque nas operações principais.

1) Laço simples

  1. Considere o algoritmo: 
    def soma_lista(nums):
    total = 0
    for x in nums:
        total += x
    return total
    Calcule o Big O de tempo em função de n = len(nums).

2) Dois laços aninhados

  1. Considere o algoritmo: 
    def pares(nums):
    cont = 0
    for i in range(len(nums)):
        for j in range(len(nums)):
            cont += 1
    return cont
    Calcule o Big O de tempo em função de n = len(nums).

3) Metade a cada iteração

  1. Considere o algoritmo: 
    def conta_divisoes(n):
    cont = 0
    while n > 1:
        n = n // 2
        cont += 1
    return cont
    Calcule o Big O de tempo em função de n.

4) Laço + busca linear

  1. Considere o algoritmo: 
    def contem_todos(a, b):
    # retorna quantos elementos de a aparecem em b
    cont = 0
    for x in a:
        if x in b:
            cont += 1
    return cont
    Assuma que a e b são listas e que o operador in faz busca linear em b. Calcule o Big O de tempo em função de n = len(a) e m = len(b).

5) Três etapas: laço, laço, laço aninhado

  1. Considere o algoritmo: 
    def rotina(nums):
    n = len(nums)
    s = 0

    # etapa 1
    for i in range(n):
        s += 1

    # etapa 2
    for i in range(n):
        s += 1

    # etapa 3
    for i in range(n):
        for j in range(i, n):
            s += 1

    return s
    Calcule o Big O de tempo em função de n = len(nums).

Entrega

  • Para cada item, escreva: (1) a contagem aproximada de operações, (2) a expressão simplificada e (3) o Big O final.
  • Opcional: classifique do menor para o maior crescimento assintótico.
